The [**ntia-conformance-checker**](https://github.com/spdx/ntia-conformance-checker)
validates SPDX SBOMs against the NTIA "minimum elements" and other baseline
requirements.

As a [**Google Summer of Code 2025**](https://github.com/spdx/ntia-conformance-checker/wiki/Adding-SPDX-3.0-Support)
contributor, I worked on adding **SPDX 3.0** support to the checker, so that
documents using the new SPDX 3 model can be validated alongside SPDX 2.
