[**licenseid**](https://github.com/bact/licenseid) is a small Python CLI and
library that takes raw license text and returns the matching
[**SPDX License Identifier**](https://spdx.org/licenses/) — useful for
automating license compliance and SBOM workflows. It is tested against a
dedicated [licenseid-dataset](https://github.com/bact/licenseid-dataset).
